Detailed Description

Implementation of the BLAKE2b cryptographic hash function.

BLAKE2b is an optimized cryptographic hash function designed for both speed and security, making it suitable for a wide range of applications. This class provides an interface to compute BLAKE2b hashes for input messages with optional keying support for keyed hashing (HMAC).

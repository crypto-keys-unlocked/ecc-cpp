std::string bytesToHex ( const std::vector< uint8_t > &  bytes)

Converts a vector of bytes (uint8_t) to a hexadecimal string.

This function takes a vector of bytes (each byte being an unsigned 8-bit integer) and converts it into a string representation where each byte is represented by its hexadecimal equivalent. For example, a byte 0xB4 would be represented as the string "B4". This is useful for displaying or storing binary data in a human-readable hexadecimal format.

Parameters
bytesThe vector of bytes to be converted to a hexadecimal string.
Returns
A std::string representing the hexadecimal value of the input bytes.

Example Usage:

std::vector<uint8_t> data = {0xBA, 0xAD, 0xF0, 0x0D};
std::string hexStr = bytesToHex(data);
// hexStr will be "BAADF00D"
